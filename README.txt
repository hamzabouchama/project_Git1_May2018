^Projet Git & GitHub - Partie 2 Activité

Cet exercice a pour objectif de mettre en pratique les notions que vous avez dans les deux premieres parties du cours sur Git et Github

Pour ce faire, vous devez créer un repository Git simple contenant :

* Un fichier nommé "README.txt" qui présente le projet
* AU moins 2 autres fichiers de votre choix (ils peuvent contenir des scripts de code, du texte sur un projet professionnel...)
* Au moins 4 commits distincts montrant l'ajout puis la modification de certains fichier
